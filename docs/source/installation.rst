============
Installation
============

Quickstart
----------

.. code-block:: bash

    pip install --upgrade syndirella
    syndirella --help

For Developers
--------------

.. code-block:: bash

    git clone [LINK]
    cd syndirella
    conda create -n syndirella python=3.10
    conda activate syndirella
    pip install -r requirements.txt

    cd aizynth
    download_public_data .
